How to use

1 Login with Github, you will be redirected to your subdomain 

http://your-subdomain.pl0t.com

2 Get API token

http://pl0t.com/api_tokens

3 Create example table, replace subdomain and API token with yours

curl \
--request POST \
--header "api_token: your-api-token" \
--data '{"rows":[[11, 12], [21, 22]]}' \
http://your-subdomain.pl0t.com/sample_table.json

4 Open table in Browser, replace subdomain in URL with yours

http://your-subdomain.pl0t.com/sample_table.json:table

5 Use API in you language.

- TypeScript Scheme full API going to be available a bit later
- Nim API

6 If you have question or need help feel free to contact
